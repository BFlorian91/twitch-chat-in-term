# twitch-chat-in-term
It's a project for parse the tchat of twitch in the terminal

## install
npm install

## Enter the name of the channel you want to track inside your term here:
file: index.js  
line 27: channels: [ 'Quin69' ]

## Start the project
npx nodemon index.js
